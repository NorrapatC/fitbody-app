Installation

ติดตั้งใน 4 ขั้นตอน

ทุกอย่างฟรี ไม่มีค่าใช้จ่าย ใช้ Supabase Free Tier + Vercel Free Tier

01
สร้าง Supabase Project

ไปที่ supabase.com → สร้าง New project ฟรี → คัดลอก Project URL และ Anon Key จาก Settings → API

supabase.com → New Project
02
รัน SQL Schema

เปิด SQL Editor ใน Supabase Dashboard → คัดลอกเนื้อหาจากไฟล์ setup.sql → กด Run เพื่อสร้างตาราง + RLS Policy

SQL Editor → Paste setup.sql → Run
03
ตั้งค่า config.js

เปิดไฟล์ js/config.js หรือใช้หน้า setup.html เพื่อ generate ไฟล์ให้อัตโนมัติ ใส่ URL และ Key ของคุณ

js/config.js → ใส่ URL + Key
04
Deploy ขึ้น Vercel

Push โค้ดขึ้น GitHub → เชื่อมต่อกับ Vercel → Auto-deploy ฟรี ไม่ต้องตั้งค่าอะไรเพิ่ม

GitHub → Vercel → Auto Deploy
